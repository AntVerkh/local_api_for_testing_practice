API Methods

GET /api/users

Retrieve all users from the system. Returns a list of user objects with their details.

GET /api/users/{id}

Retrieve a specific user by their ID. Returns the user object if found.

POST /api/users

Create a new user. Requires user details in the request body. Returns the created user.

PUT /api/users/{id}

Update an existing user. Requires user ID and updated details in the request body.

DELETE /api/users/{id}

Delete a user by their ID. Returns success status after deletion.

GET /api/users/search

Search users by various criteria. Supports filtering by name, email, and other attributes.
